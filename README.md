## Описание проекта

**Cafe Order Manager** — это приложение, разработанное для автоматизации работы небольшого кафе. Система позволяет эффективно управлять заказами, отслеживать занятость столиков, вести расчет стоимости и контролировать меню.

Проект создан для упрощения работы персонала: официанты могут быстро принимать заказы и печатать чеки, а администраторы — управлять составом блюд и ценами.

## Основные возможности
- **Управление меню**: добавление, удаление и редактирование блюд (название, цена, категория).
- **Работа со столиками**: отслеживание статуса (свободен/занят).
- **Создание заказов**: привязка заказа к столику, добавление блюд из меню.
- **Расчёт стоимости**: автоматический подсчёт суммы заказа с учётом выбранных позиций.
- **Статусы заказов**: контроль жизненного цикла заказа (открыт, закрыт, оплачен).
- **Разделение счёта**: возможность разделить итоговую сумму на несколько частей.
- **Печать чеков**: генерация текстового файла с детализацией заказа для выдачи гостю.
- **Распределение прав доступа**: разграничение прав доступа для официантов и администраторов.

## Установка и использование

Следуйте этим простым шагам, чтобы запустить проект локально.

### Предварительные требования
- Python 3.8 или выше.
- Менеджер пакетов `pip`.

<a id="instructions"></a>
### Инструкция по установке

1.  **Клонируйте репозиторий**
    ```
    git clone https://github.com/your-username/cafe-order-manager.git
    ```
2.  **Перейдите в директорию проекта**
    ```
    cd cafe-order-manager
    ```
3.  **(Рекомендуется) Создайте виртуальное окружение**
    ```
    python -m venv venv
    source venv/bin/activate  # Для Linux/Mac
    # venv\Scripts\activate  # Для Windows
    ```
4.  **Установите зависимости**
    В проете есть файл `requirements.txt` (например, для GUI библиотек), выполните:
    ```
    pip install -r requirements.txt
    ```
5.  **Запустите приложение**
    ```
    python main.py
    ```

## Сценарии использования

### Для официанта
1.  **Авторизация** в системе.
2.  Просмотр списка свободных столиков.
3.  **Открытие заказа**: выбор столика и добавление блюд из меню.
4.  Внесение изменений в открытый заказ (добавление/удаление позиций).
5.  **Разделение счёта** по запросу гостей (указание количества частей или сумм).
6.  **Закрытие заказа** и печать чека в текстовый файл.
7.  Освобождение столика.

### Для администратора
1.  **Авторизация** с правами администратора.
2.  **Управление меню**:
    - Добавление нового блюда (название, цена, категория: "Основное", "Напитки", "Десерты").
    - Редактирование цены или названия существующего блюда.
    - Удаление блюда из меню.
3.  Просмотр всей статистики (опционально).
