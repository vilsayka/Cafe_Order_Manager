## Описание проекта

**Cafe Order Manager** — это консольное приложение (или приложение с GUI, в зависимости от реализации), разработанное для автоматизации работы небольшого кафе или ресторана. Система позволяет эффективно управлять заказами, отслеживать занятость столиков, вести расчет стоимости и контролировать меню.

Проект создан для упрощения работы персонала: официанты могут быстро принимать заказы и печатать чеки, а администраторы — управлять составом блюд и ценами.

### Основные возможности
- **Управление меню**: добавление, удаление и редактирование блюд (название, цена, категория).
- **Работа со столиками**: отслеживание статуса (свободен/занят).
- **Создание заказов**: привязка заказа к столику, добавление блюд из меню.
- **Расчёт стоимости**: автоматический подсчёт суммы заказа с учётом выбранных позиций.
- **Разделение счёта**: возможность разделить итоговую сумму на несколько частей (например, для компании гостей).
- **Статусы заказов**: контроль жизненного цикла заказа (открыт, закрыт, оплачен).
- **Печать чеков**: генерация текстового файла с детализацией заказа для выдачи гостю.
- **Ролевая модель**: разграничение прав доступа для официантов и администраторов.

## Установка и использование

Следуйте этим простым шагам, чтобы запустить проект локально.

### Предварительные требования
- Python 3.8 или выше.
- Менеджер пакетов `pip`.

### Инструкция по установке

1.  **Клонируйте репозиторий**
    ```bash
    git clone https://github.com/your-username/cafe-order-manager.git
    ```
2.  **Перейдите в директорию проекта**
    ```bash
    cd cafe-order-manager
    ```
3.  **(Рекомендуется) Создайте виртуальное окружение**
    ```bash
    python -m venv venv
    source venv/bin/activate  # Для Linux/Mac
    # venv\Scripts\activate  # Для Windows
    ```
4.  **Установите зависимости**
    Если в проекте есть файл `requirements.txt` (например, для GUI библиотек), выполните:
    ```bash
    pip install -r requirements.txt
    ```
5.  **Запустите приложение**
    ```bash
    python main.py
    ```
